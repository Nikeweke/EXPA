### Начало

Подключить в `<head>` Vue.js и свой `app.js`
```html
<!-- VUE.js -->
<head>
   <script src="https://unpkg.com/vue" charset="utf-8"></script> <!-- must be in head -->
</head>

...

  <script src="app.js" charset="utf-8"></script> <!-- must be before close body tag -->
</body>
```
